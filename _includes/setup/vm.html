Virtual Machine

While it is better for everyone to install tools in your working environemnt, sometimes installation hits unexpected obstacles. So that you can continue to participate in a lesson even if one of your software programs fails, we provide a Linux virtual machine contains all the necessary software pre-installed. Please install VirtualBox and download this virtual machine image. Load the VM into VirtualBox by doing Import Appliance and loading the .ova file. The ova file is 2Gbytes, and isn't anything special--it is just a linux environment in which git and python work that we have as a backup.
